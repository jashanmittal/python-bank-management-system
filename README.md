A console-based Bank Management System built using Python and Object-Oriented Programming.

This project allows users to create bank accounts, securely log in, deposit and withdraw money, transfer funds, change passwords, and maintain transaction history with JSON data persistence.

## Features

- Create Bank Account
- Secure Login
- Deposit Money
- Withdraw Money
- Transfer Money Between Accounts
- Balance Checking
- Transaction History
- Change Password
- Unique Account Numbers
- JSON Data Storage
- Input Validation

- ## Technologies Used

- Python 3
- JSON
- Object-Oriented Programming
- datetime module
- random module

- ## Concepts Practiced

- Classes & Objects
- Constructors
- Methods
- Lists of Objects
- Loops
- Conditionals
- Exception Handling
- File Handling
- JSON Serialization
- Data Persistence

- ## How to Run

1. Clone the repository

git clone https://github.com/YourUsername/python-bank-management-system.git

2. Open the project folder.

3. Run

python bank.py
